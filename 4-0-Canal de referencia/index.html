4. Canal de referencia

Para toda la elaboración del proyecto, además de utilizar librerías oficiales de las herramientas empleadas, se utilizaron videos de diversos canales de youtube. Sin embargo, para toda la conceptualización general de conocimientos para el proyecto, se empleó el siguiente canal de youtube:

Particularmente, se utilizó la siguiente la playlist Building a mobile robot.

En este canal se realiza la simulación y construcción de un robot diferencial; claramente al compararlo con el proyecto presentado tienen objetivos muy diferentes, pero al llevarse con las mismas herramientas y hasta por el mismo _camino_, se decidió tomar de referencia como conceptualización y hasta entrenamiento para la elaboración del proyecto principal.
Según esto, inicialmente se realizó el mismo proceso mostrado en el canal con el robot diferencial. Con esto realizado, se repitiría el mismo proceso pero con un servomotor; se tomó este enfoque porque el SCARA está construido con _motores de paso_ con comportamiento similar a los servomotores; por lo que, al realizar el control con un servomotor, bastaría con _adaptar_ el proyecto al SCARA.
